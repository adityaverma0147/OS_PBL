About This Server

This web server is a lightweight custom-built HTTP server written in C. It serves static HTML pages, handles requests efficiently, and helps understand how modern servers like Apache or Nginx work under the hood.

The project demonstrates low-level socket programming, threading, and HTTP protocol parsing — all packed in a minimal yet educational setup.
